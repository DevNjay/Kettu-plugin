**ROOT CAUSE**: 
Plugin was using native HTML heading tags (h2, h3, h4) which are not compatible 
with Kettu's React Native-based rendering system.

**FIX APPLIED**: ✅
Replaced all native heading tags with styled div elements:
- h2 → div with fontSize: 1.5em
- h3 → div with fontSize: 1.1em  
- h4 → div with fontSize: 1em

**WHAT WAS THE PROBLEM?**

Kettu uses React Native, which doesn't support native HTML heading elements 
(h1-h6). It needs either:
1. Divs with styling (what we did)
2. Custom Kettu UI components
3. Primitive React Native components

Native heading tags don't exist in React Native's component model, causing 
the "config getter callback" error.
